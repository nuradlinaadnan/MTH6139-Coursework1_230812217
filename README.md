# MTH6139-Coursework1_230812217
A time series analysis of How interested the UK is in Malaysia using R and Meta's Prophet. This project explores how much interest people in the UK have shown in Malaysia over the past two decades as measured by Google search activity using data from Google Trends.
A live version of this analysis can be viewed here: https://mth6139-coursework1-280812217.netlify.app/
